Navigation Component Demo

This page demonstrates the navigation component with full accessibility support.

Features

  • Responsive sidebar navigation
  • Collapsible sub-menus
  • Mobile navigation drawer
  • Keyboard navigation (Arrow keys, Home, End, Enter, Space)
  • ARIA attributes for screen readers
  • Notification badges
  • Active item tracking
  • State persistence (localStorage)
  • Focus management and focus trap
  • Smooth animations and transitions

Keyboard Shortcuts

↑ / ↓
Navigate between menu items
Home / End
Jump to first/last menu item
Enter / Space
Expand/collapse sub-menus or activate links
← / →
Collapse/expand sub-menus when focused
Esc
Close mobile menu
Tab
Navigate between focusable elements

JavaScript API

// Initialize navigation
const nav = new Navigation({
  sidebarSelector: '[data-nav-sidebar]',
  toggleSelector: '[data-nav-toggle]',
  scrollToActiveOnInit: true,
  persistState: true
});

// Event listeners
nav.sidebar.addEventListener('nav:expand', (e) => {
  console.log('Menu expanded:', e.detail.item);
});

nav.sidebar.addEventListener('nav:activate', (e) => {
  console.log('Item activated:', e.detail.item);
});

// Methods
nav.expand(item);       // Expand menu item
nav.collapse(item);     // Collapse menu item
nav.setActiveItem(item); // Set active item
nav.toggleCollapse();   // Toggle sidebar collapse
nav.destroy();          // Clean up
