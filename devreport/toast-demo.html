Usage Examples

Basic Usage

// Show a simple toast showToast('Hello World!', 'info'); // Show success toast showSuccess('Operation completed!'); // Show error toast showError('Something went wrong!'); // Show warning toast showWarning('Be careful!'); // Show info toast showInfo('Did you know?');

Advanced Options

// Toast with custom duration showSuccess('Saved!', 3000); // Toast with title showSuccess('Your profile has been updated', { title: 'Success!', duration: 5000 }); // Persistent toast (no auto-dismiss) showError('Critical error occurred', { title: 'Error', duration: 0 }); // Toast with custom position showInfo('New message received', { position: 'bottom-right' }); // Dismiss specific toast const toastId = showSuccess('Hello'); dismissToast(toastId); // Dismiss all toasts dismissAllToasts(); // Change global position setToastPosition('top-center');
