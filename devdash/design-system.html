Usage Guidelines

  • Use semantic color names (primary, success) not specific colors (blue, green)
  • Always test color combinations for accessibility compliance
  • Use consistent spacing increments from the scale
  • Apply elevation systematically to establish hierarchy
  • Test all interactions across breakpoints
